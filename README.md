# Random Quote Generator

A full-stack Random Quote Generator built with Python Flask and vanilla JavaScript. This application allows users to get inspired by random quotes from various categories.

## Features

- ✨ Get random quotes with a single click
- 📚 15+ curated quotes across multiple categories
- 🎯 Filter quotes by category
- 📋 Copy quotes to clipboard with one click
- 🐦 Share quotes on Twitter
- ⌨️ Keyboard shortcuts (Space/N for new quote, C for copy)
- 📱 Responsive design for all devices
- 🔄 Real-time quote updates
- 💫 Smooth animations and transitions

## Technology Stack

- **Backend**: Python 3.x with Flask
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: RESTful API with JSON responses
- **Styling**: Custom CSS with gradient backgrounds
- **Tools**: Flask-CORS for cross-origin support
